# Payroll Management System (JavaFX)

## Description
A JavaFX-based Payroll Management System that allows users to:
- Capture employee details
- Calculate salaries and deductions
- Generate payslips
- Store payroll data in a database

## Technologies Used
- Java
- JavaFX
- Scene Builder
- IntelliJ IDEA

## Project Structure
src /

├── main /

│ ├── java /

│ └── resources /

## How to Run
1. Open the project in IntelliJ IDEA
2. Ensure JavaFX is configured
3. Run `Launcher.java`
